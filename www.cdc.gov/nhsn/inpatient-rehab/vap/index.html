Surveillance for Ventilator-Associated Pneumonia (VAP) Events

Recommend on Facebook Tweet ShareCompartir

NOTE: PNEU/VAP surveillance is available in-plan for patients of any age in non-NICU pediatric locations. For in-plan surveillance conducted for mechanically-ventilated patients in adult locations (regardless of age), use the Ventilator-associated Event (VAE) protocol (see Ventilator-Associated Event (VAE) Protocol, January 2017 )
