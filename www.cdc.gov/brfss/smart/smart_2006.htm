Data Files

The data files are provided in both ASCII and SAS Transport formats. There are 216,379 records in the county data set and 250,584 records in the MMSA data set. BRFSS and BRFSS GIS Maps data and documentation are also available for this data year.

2006 SMART BRFSS County Data (ASCII) [ZIP - 14 MB]
Data released August 18, 2011
This file is in ASCII format. It has a fixed record length of 612 positions.

2006 SMART BRFSS County Data (SAS Transport Format) [ZIP - 26 MB]
Data updated August 18, 2011
This file was exported from SAS V9.2 in the XPT transport format. This file contains 332 variables. This format can be imported into SPSS or STATA. Please note: some of the variable labels become truncated in the process of converting to the XPT format so they may be slightly different from what is on the County_SASOUT06.SAS program.

County Variable Layout [PDF - 68 KB]
Format information on variable name by column position.

2006 SMART BRFSS MMSA Data (ASCII) [ZIP - 15 MB]
Data released August 18, 2011
This file is in ASCII format. It has a fixed record length of 662 positions.

2006 SMART BRFSS MMSA Data (SAS Transport Format) [ZIP - 27 MB]
Data released August 18, 2011
This file was exported from SAS V9.2 in the XPT transport format. This file contains 331 variables. This format can be imported into SPSS or STATA. Please note: some of the variable labels become truncated in the process of converting to the XPT format so they may be slightly different from what is on the MMSA_SASOUT06.SAS program.

MMSA Variable Layout [PDF - 68 KB]
Format information on variable name by column position.
This file is in ASCII format. It has a fixed record length of 662 positions.

SAS Resources

The following files are included for use with SAS software developed by SAS Institute, Inc. The SAS files are programs that can be read using any text editor. Save SAS files as plain text files.

Format06.sas [SAS - 125 KB]
SAS for Windows program used to generate the 2006 format library.

format06 [SAS7BCAT - 517 KB]
Format library used with the 2006 BRFSS SAS Data File.

MMSA06_SASOUT.SAS [SAS - 29 KB]
SAS for Windows program to convert the MMSA ASCII file into a SAS dataset.

CNTY06_SASOUT.SAS [SAS - 29 KB]
SAS for Windows program to convert the County ASCII file into a SAS dataset.

transprt.sas [SAS - 3 KB]
SAS for Windows program to convert the XPT (SAS Transport) file into a SAS dataset.

Formas06.sas [SAS - 9 KB]
SAS for Windows format assignment statements.
