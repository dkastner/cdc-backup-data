2012 BRFSS Landline Telephone Survey Data including Optional Modules

Recommend on Facebook Tweet ShareCompartir

In 2012, the Behavioral Risk Factor Surveillance System (BRFSS) consisted of two survey modes, landline telephone and cellular telephone. Several states collected optional modules only on the landline telephone survey. The optional modules were asked throughout all 12 months of the survey in order to be included in this aggregate file. This additional data set containing the optional module data and weights is provided for use with variables resulting from questions asked only on the landline telephone version of the questionnaire.

For more information, please refer to the documentation below.

About the Landline Questionnaire Module Data

Use of BRFSS Landline Multiple Version Questionnaire Data [PDF - 189 KB]

Please read this document before performing any data analyses. This document contains a list of states that collected optional modules on their landline survey and the modules they used.
