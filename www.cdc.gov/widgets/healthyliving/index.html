How Healthy Is Your State?

This widget provides users with a quick and easy way to see state data related to cholesterol, diabetes, heart attacks, high blood pressure, nutrition, obesity, physical activity, stroke, and tobacco use. The widget uses Behavioral Risk Factor Surveillance System (BRFSS) data from 2011 to 2014 for all 50 states and the District of Columbia.

National Data: To embed the widget with national data, click the "Share" button; then, copy and paste the code into your website or blog.

State-specific Data: To customize the widget to display data for a specific state: 1) select the state from the dropdown menu, 2) click the "Share" button, and 3) copy the code from the widget and paste it into your website or blog.

Public Health Image Library (PHIL) Image of the Day (IOTD)

Add the Public Health Image Library Daily Widget to your Web site to receive a different image each day from the CDC library. The Public Health Image Library (PHIL) is an electronic gateway to CDC's pictures; depicting a myriad of topics from the realm of public health, including disease processes, pathogens, and the history of public health and the CDC. Web users will click this widget and find a larger sized version of the image of the day with more information, as well as a link to a print-quality version for publication purposes.
