State Tobacco Activities Tracking and Evaluation (STATE) System

Recommend on Facebook Tweet ShareCompartir

Custom Reports

Use Custom Reports to build your own report by selecting the topics, states, and years of data.

View Data results in the System or export to Excel.

State Highlights

Use the State Highlights report to see a variety of data from across the system for a single selected state.

View highlights in the System or export to PDF.

Interactive Maps

Use Interactive Maps to access key data from across the System in a US map and data table.

View map and table in the System or export a table to Excel.

Latest Updates

January 2017

New Features added to STATE System:
• Redesigned homepage to allow easier access to fact sheets, slide decks, quick start guides, and other content.
• Added enhancements to Custom Reports and State Highlights to increase usability of the reports.
