Update to 2005 Vermont Birth Data

Recommend on Facebook Tweet ShareCompartir

State specific data for Vermont in the 2005 natality file and presented in the report Births: Final Data for 2005 [PDF - 2.5 MB] are based on an incomplete file of Vermont birth records; the total number of Vermont resident births is under-reported by about 3 percent. Information based on the complete count (6,473) of resident births can be accessed below.

 

Vermont-specific Updates

Table V-1 updates Vermont-specific data presented in Births: Final Data for 2005 [PDF - 2.5 MB] based on the complete file of Vermont births.

Additional tabulations based on the complete file of Vermont resident births using all items on the 2005 public use natality file may be made using the VitalStats online data access tool.

For additional information, please contact births@cdc.gov or call (301) 458-4111.
