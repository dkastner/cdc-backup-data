Home

Welcome to the NHANES-CMS Linked Data Web Tutorial!  The National Center for Health Statistics (NCHS) developed this tutorial to meet the growing needs of NHANES-CMS linked data users.

This tutorial is one of several NHANES Tutorials that have been developed to promote broader and more proficient use of NHANES data.

 

Before you get started:

  • Check out the Objectives to orient yourself to the tutorial’s content.

  • Read the Introduction to find answers to frequently asked questions about NHANES-CMS Linked Data and this tutorial.

  • Go to Knowledge & Skills for information about the basic knowledge and skills required to use this tutorial as a self-learning tool.

  • Go to Technical & Software Requirements for information about what’s required to view the tutorials correctly and run the sample programs properly.  This section also is the place to go if you need help with technical problems.
