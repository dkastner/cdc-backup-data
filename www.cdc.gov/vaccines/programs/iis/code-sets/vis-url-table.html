Vaccine Information Statement (VIS) to Current Uniform Resource Locator (URL) Table

Recommend on Facebook Tweet ShareCompartir

Get Email Updates

CDC has developed a number of tables related to VIS documents: the VIS Lookup Table and the CVX-VIS Mapping Table.  These tables were developed to support and take advantage of the 2D data matrix barcodes on the VIS documents.  There are two fields contained within the 2D barcode on the VIS.  The first field is a code that identifies the VIS document type (MMR, seasonal influenza, anthrax, etc.). This field uses the GS1 Global Document Type Identifier (GDTI).  The second field is the edition or publication date of the VIS. The CVX-VIS mapping table shows the relationship between the vaccine’s CVX codes and the VIS codes.  For additional information see the CVX tables and the VIS Lookup table.

This table, the VIS to Current URL Table, contains the URL links to the corresponding VIS document and was developed to assist developers to automatically generate the current version of the VIS.  The table contains the URL links to both the Portable Document Format (PDF) version of the VIS as well as the HyperText Markup Language (HTML) version. The URL links are for the most up-to-date or current VIS document. The HTML format of this table (below) as well as the PDF and XML versions contain the following data items.

  • VIS Document Type Description – Concept Name
  • VIS GDTI document code / Concept code - (Global Document Type Identifier)
  • VIS PDF URL
  • VIS HTML URL

The Excel file version contains the above data items as well as additional columns to allow users to directly link to the respective PDF and HTML web pages.

Note: The VIS Lookup Table contains both the VIS Fully-Encoded Text String as well as the GDTI. The VIS Fully-Encoded Text String is used for electronic reporting purposes. The VIS Lookup Table allows users to link the VIS Fully-Encoded Text String to the corresponding GDTI.
