Below are the items that were at one time listed in the Spotlight News box on the IIS home page. This list serves as an archive and assists those that come to this site and need to know what has been added or revised on the site.

 

List of New and Updated Items (By Date)

  • VaxChora® and Flucelvax® Quadrivalent added to vaccine, trade name, vaccine group and NDC tables (6/21/2016)
  • VTrckS ExIS resource: Strategies for Improving Accountability, Efficiency and Provider Satisfaction (10/8/2015)
  • New code set: VIS to Current URL Table (9/1/2015)
  • IIS list of publications has been updated in the searchable database (8/20/2015)
  • HL7 Standard Code Set: CVX-VIS Mapping Table (8/17/2015)
  • Archived Meaningful Use 3 Tables for Certification (4/16/2015)
  • IIS list of publications has been updated in the searchable database. (8/15/2014)
  • Announcement: Recommendation Regarding Increasing Vaccination Rates Through Use of Immunization Information Systems — Community Preventive Services Task Force - The Community Preventive Services Task Force recently posted new information on its website: "Increasing Appropriate Vaccination: Immunization Information Systems." (6/12/2014)
  • AIRA Education Webinar: Under the Hood of the HL7 V2.5.1 IG, Release 1.5, Session 2 - April 14, 2014 3:00 pm ET. Rob Savage will conduct the second of two sessions on the IG update and answer questions. (4/8/2014)
  • CDSi: Logic Spec for ACIP Rec, version 1.6 (3/26/2014)
  • Immunization Information Systems: A Decade of Progress in Law and Policy - A new journal article with findings and analysis of CDC’s IIS legislative survey. (3/12/2014)
  • AIRA Education Webinar: HL7 Part 2, Implementation Guide Update Review, Session 1 - March 25, 2014 4:00 pm ET. Rob Savage will present a review of the IG update and answer questions. (3/5/2014)
  • Progress in Immunization Information Systems, United States - 2012 - Results from the 2012 IIS Annual Report, including updates on child and adult participation in IIS and progress towards functional standards. (12/13/2013)
  • AIRA Webinar: IIS Standards Harmonization - December 17, 2013 3:00pm-4:00pm ET. Nathan Bunker and Mary Beth Kurilo will present on the recent Status Check validation project and IIS standards development. (12/11/13)
  • 2D Vaccine Barcodes - Explains need for and potential impact of 2D barcodes on EHRs and IIS, describes CDC's 2D barcode pilot project, and provides links to additional resources for manufacturers and vaccine providers (4/2/13)
  • VIS Barcode Lookup Table Shows the codes captured when scanning barcodes on vaccine information statements (4/2/13)
  • Q&A About IIS Sentinel Sites Q&As now reflect changes in grantees, staffing and project objectives implemented at start of new cooperative agreement cycle. (3/1/2013)
  • Progress in IIS - U.S., 2011 The January 25, 2013, MMWR documents results from the 2011 IIS Annual Report and includes updates on child participation in IIS and core data element completion. (1/28/2013)
  • IIS Functional Standards, 2013 - 2017 Revised IIS Functional Standards for implementation by CDC-funded Immunization Programs during the period 2013 to 2017.(12/12/12)
  • Logic Specification for ACIP Recommendations A single, implementation-neutral foundation for developing and maintaining CDS engines (10/29/12)
  • Two-Dimensional (2D) Vaccine Barcoding Offers information on the need for and potential impact of 2D barcoding on EHRs and IIS, describes CDC's 2D barcode pilot project, and provides links to additional resources (10/25/12)
  • Meaningful Use Stage 2 Final Rule The final rules for Meaningful Use Stage 2 were published in the Federal Register on September 4, 2012. (9/5/12)
  • IISAR Data Participation Rates and Maps National child, adolescent, and adult participation rates and maps are now updated with CY2010 Immunization Information Systems Annual Report (IISAR) data. (8/31/12)
  • HL7 Version 2.5.1 Implementation Guide for Immunization Messaging, Release 1.4 (8/1/12)
  • The MIROW web page with information on IIS operational best practice guidelines has been updated, and MIROW's new guide on IIS inventory management operations is now available. (7/30/12)
  • Progress in Immunization Information Systems — United States, 2010 The June 29, 2012, MMWR documents results from the 2010 IIS Annual Report including updated child and provider site participation rates (6/28/12)
  • Transport Layer Webinars Check out the new webinars providing programmatic overview and technical details for implementers (5/17/12)
  • Source Code Samples Now Available - Source code examples of the SOAP IIS Messaging client for .Net Client and Java Client Server Projects (5/17/12)
  • Reference Implementation EHR – See the new reference implementation documents to help with testing for implementers (5/17/12)
